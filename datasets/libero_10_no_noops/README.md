# LIBERO-10 RLDS/TFDS Data

Place the LIBERO-10 TFDS/RLDS files under this directory:

```text
datasets/libero_10_no_noops/1.0.0/
```

The local release package used during preparation includes the TFRecord shards in this location. They are not tracked in Git because the full dataset is several GB and exceeds normal GitHub repository limits. The included feature file is sufficient to run LED scoring and subset selection.
